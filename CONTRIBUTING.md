## Cara Contribute

1. Fork repository ini
2. Clone fork kamu: `git clone https://github.com/username/interactive-ui-js.git`
3. Buat branch baru: `git checkout -b feature/fitur-keren`
4. Commit changes: `git commit -m 'Add: fitur keren'`
5. Push ke branch: `git push origin feature/fitur-keren`
6. Buka Pull Request

## Code Style

- Gunakan 4 spaces untuk indentation
- Camelcase untuk variable names
- Comment untuk logic yang kompleks
- Keep functions small & focused
